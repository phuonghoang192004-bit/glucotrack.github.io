📊 Kết nối Google Sheets

Dữ liệu của bạn sẽ lưu lên Google Sheets — mỗi người dùng một trang tính riêng. Cần thiết lập Apps Script một lần duy nhất.

📋 Hướng dẫn (5 phút, 1 lần):
  1. Mở sheets.new → tạo spreadsheet mới, đặt tên GlucoTrack
  2. Vào menu Extensions → Apps Script
  3. Xoá code mẫu, nhấn xem & copy code Apps Script
  4. Dán vào editor → Deploy → New deployment
  5. Type: Web app · Execute as: Me · Access: Anyone
  6. Copy Web app URL → dán vào bước tiếp theo
